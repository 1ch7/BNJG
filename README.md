# Challenges and Approach
We chose track 3 which aims to create a digital solution that helps optimize usage and reduce unnecessary waste because in Malaysia, we have seen so many restaurants unable to manage and allocate their resources efficiently which lead to waste. We help those restaurants by doing proactive discount on the overstocked items.

# Instructions 
1. Download Sample Data.zip from files in the git repo and extract it
2. Launch the web app
3. Make an account
4. Import the Cookbook from Sample Data
5. For the daily sales, import the report from Sample Data based on the day
